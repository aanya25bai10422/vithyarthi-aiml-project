💰 Expense Tracker (Python CLI Project)
A simple command-line Expense Tracker built using Python that helps users manage their daily expenses efficiently. This project allows users to add, view, edit, and delete expenses, with all data stored in a JSON file for persistence. 
📌 Project Title 
Smart Expense Tracker 
📖 Overview
The Smart Expense Tracker is a simple Python command-line application that helps users manage daily expenses by adding, viewing, editing, and deleting records. All data is stored in a JSON file, ensuring it is saved and accessible anytime. It is designed to demonstrate basic Python concepts like file handling and user interaction. 
📌 Features 
•	Add new expenses with category  
•	View all saved expenses 
•	Edit existing expenses 
•	Delete expenses
•	 Automatic saving using JSON file
•	 Persistent storage (data saved even after closing program) 
🛠️ Technologies / Tools Used 
•	Python 3 – Core programming language
•	 JSON – Data storage for saving expenses
•	 Command Line Interface (CLI) – User interaction
•	 Built-in Python Functions – File handling, input/output operations

▶️ How to Use
Once you run the program, you will see a menu:
Expense Tracker
1. Add Expense
2. View Expenses
3. Edit Expense
4. Delete Expense
5. Exit

🔹 Add Expense
•	Enter: 
o	Name 
o	Amount 
o	Date (YYYY-MM-DD) 
o	Category (Food, Transport, etc.) 

🔹 View Expenses
•	Displays all saved expenses like: 
1.	2026-03-26 - Lunch: Rs 200[Food]
🔹 Edit Expense
•	Select expense number 
•	Update details (leave blank to keep old values)
🔹 Delete Expense
•	Select expense number to remove it
🔹 Exit
•	Saves all data and exits program

•	All expenses are stored in: 
expenses.json
